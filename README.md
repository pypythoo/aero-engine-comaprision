# aero-engine-comaprision

🌐 Live Website: https://aeroengcomp-o4nfvkd4.manus.space/

This is an educational aerospace engineering project focused on understanding and comparing real aircraft propulsion technologies from major industry players such as GE Aerospace, Rolls-Royce and Safran.

## Purpose

The goal of this project was to learn about modern turbofan engines, propulsion system performance factors and future aviation technologies while also exploring AI-assisted web development workflows.

## How this project was built

- Initial website structure and UI were generated using AI website builder tools.
- Content direction, aerospace topic research and learning sections were manually studied and refined by me as an Aerospace Engineering student.
- The project focuses on simplifying complex propulsion concepts for beginner learners.

## Topics Covered

- Commercial turbofan engines (GE9X, Trent XWB, LEAP)
- Fighter jet propulsion basics
- Thrust generation and bypass ratio concepts
- Future propulsion technologies such as hybrid-electric systems
- Aerospace materials innovation

## Disclaimer

This is an independent educational student project.  
It is not affiliated with GE Aerospace, Rolls-Royce or Safran.

## Learning Outcome

This project helped me combine aerospace curiosity with modern AI tools and understand how engineering research can be communicated through digital platforms.
